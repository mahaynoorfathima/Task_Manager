ABOUT US:

It is a web application , mainly based on company environment. Consist of two login pages , where 0ne is for employee and another is for manager.In manager portal we create two sub sections, Then in employee portal there will be section which displays the tasks assigned by the manager.Another section for accept or reject the task .

If he/she reject the task then a new page will open where he/she should give the specified reason for the rejection of the task .This message will be displayed on managers 2 sub section , and according to that he will take corrective actions.

Task manager provides a system for better handling the team, get notified in multiple ways Team work allocation/ task monitoring can be managed from mobile even while traveling And fast update of the work. It is a platform empowers managers and team members to assign and receive tasks, add followers, consolidate comments in a single point, organize projects and access evidence-based performance evaluations This project uses database mysql ,which allows to store the wide range of details and new updates . the front end of the website is designed by using html, bootstrap, css and javascript while backend is designed with php.
